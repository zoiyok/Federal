关于规范暑假作息时间的通知
发布方：新兰合众国行政院   [2022] 第1号   发布时间：2022年1月11日
为确保寒假时间充分利用，行政院提出2022寒假作息方案并取得表决通过。方案内容如下：
( 一 ) 每日学习时间为6小时，其中上午、下午、晚上各为2小时。
( 二 ) 学习内容分别为：美国大学生数学建模竞赛、蓝桥杯嵌入式设计大赛、中国大学生计算机设计大赛。
( 三 ) 优先级为嵌入式>美赛>计设，如遇冲突则按照此顺序进行保留。
( 四 ) 各个时间段进行计时，除非特殊情况，否则必须达标。如不达标，则对次日运势加以惩戒。
